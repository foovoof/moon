# Standards — معايير منصة Moon

هذا المجلد هو **المصدر الرسمي للحقيقة (Single Source of Truth)** لكل
التعريفات المفاهيمية والبيانية المستخدمة عبر منصة Moon. أي وحدة كود
(`domain/`, `contracts/`, `graph/`, ...) يجب أن تلتزم بما هو معرَّف هنا.

## البنية

| المجلد | المحتوى |
|--------|---------|
| `ontology/` | الأنطولوجيا المفاهيمية: تعريف الكيانات وعلاقاتها على مستوى مجرد |
| `taxonomy/` | تصنيفات هرمية (تصنيف المصادر، تصنيف المخاطر، تصنيف الأدلة...) |
| `concepts/` | مفاهيم مشتركة عابرة للمجال (مثل "الهوية"، "الثقة"، "الحدث") |
| `entities/` | كتالوج أنواع الكيانات المعتمدة (شخص، منظمة، جهاز، حساب، مركبة...) |
| `relationships/` | كتالوج أنواع العلاقات المعتمدة بين الكيانات |
| `confidence/` | نموذج درجات الثقة (Confidence Scoring) الموحّد لكل نتيجة/دليل |
| `evidence/` | معايير توثيق الأدلة، سلسلة الحفظ، قابلية الإثبات |
| `lifecycle/` | دورة حياة الحالة/التحقيق/الدليل من الفتح إلى الأرشفة |
| `classifications/` | مستويات التصنيف الأمني للمعلومات (عام/سري/سري للغاية) |
| `metadata/` | حقول البيانات الوصفية المشتركة (المصدر، الطابع الزمني، الإصدار...) |
| `events/` | كتالوج الأحداث (`catalog.yaml`)، قواعد التسمية، إصدار الأحداث |
| `data_dictionary/` | قاموس البيانات التفصيلي حقلًا بحقل لكل كيان مجال |

## كيفية إضافة معيار جديد

1. حدّد الفئة الأنسب من الجدول أعلاه (أو أنشئ فئة جديدة إذا لزم بموافقة الحوكمة).
2. أضف تعريفًا واضحًا (YAML أو Markdown حسب طبيعة المحتوى).
3. إن كان المعيار يمثل كيان مجال جديد، أضف ملفًا مقابلًا في
   `standards/data_dictionary/<entity>.md` بنفس القالب المستخدم للكيانات
   الحالية (identity, entity, relationship, evidence, source, finding,
   report, investigation, workflow, event).
4. حدّث `domain/` و `contracts/dto/` لتعكس التعريف الجديد.
5. أضف اختبار عقد في `tests/contract/` يثبّت التوافق.

## مبدأ التوافق مع الإصدارات السابقة

أي تعديل على معيار قائم (حذف حقل، تغيير نوعه) يُعتبر **تغييرًا كاسرًا
(Breaking Change)** ويتطلب:
- ترقية الإصدار الرئيسي في `standards/events/event_versioning.yaml` (إن
  كان الحدث متأثرًا).
- توثيق التغيير في `CHANGELOG.md` وربطه بسجل قرار في `architecture/adr/`.
